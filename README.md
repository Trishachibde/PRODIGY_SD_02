# PRODIGY_SD_02
Task 02 - Software Development Internship

This project is a Number Guessing Game developed using Python. The program generates a random number between 1 and 100, and the user has to guess the correct number. After each guess, the program provides hints indicating whether the guessed number is too high or too low. The game continues until the correct number is guessed, and the total number of attempts is displayed at the end.

This project helped me improve my understanding of loops, conditional statements, user input handling, and Python's random module while creating an interactive console-based application.
